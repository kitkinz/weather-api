## Installation
Follow these steps to run this application:
1. Clone this repository:
```
git clone https://github.com/kitkinz/expense-tracker.git
```
2. Navigate to the project directory:
```
cd ExpenseTracker
```
3. Restore dependencies:
```
dotnet restore
```
4. Build the project:
```
dotnet build
```
5. Run the app:
```
dotnet run
```
